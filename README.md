# GroupDocs.Watermark - Documentation Source Files

This repository contains Markdown files used to generate [GroupDocs.Watermark](https://docs.groupdocs.com/watermark/) static documentation.

## About GroupDocs.Watermark

[GroupDocs.Watermark](https://products.groupdocs.com/watermark/) is a watermark management SDK that enables you to add, search, and remove watermarks from documents in popular file formats.

Available for .NET, Java, Node.js, and Python.

## Repository Purpose

The files in this repository are used to:

- Build and maintain the official [documentation](https://docs.groupdocs.com/watermark/).
- Ensure documentation is always up-to-date with the latest product release.
- Provide a consistent structure for automated builds and publication.

## How to Use

This repository is not meant for direct use, as it contains source files for generating static content with Hugo.
