This section covers the everyday watermarking tasks you can accomplish with just a few lines of code:

- [Add text watermarks]({{< ref "watermark/python-net/developer-guide/basic-usage/add-text.md" >}}) — stamp styled text onto a document.
- [Add image watermarks]({{< ref "watermark/python-net/developer-guide/basic-usage/add-image.md" >}}) — overlay a logo or graphic.
- [Adding repeated watermarks]({{< ref "watermark/python-net/developer-guide/basic-usage/adding-repeated-watermarks.md" >}}) — tile a watermark across the whole page.
- [Customize watermarks]({{< ref "watermark/python-net/developer-guide/basic-usage/customize.md" >}}) — control color, font, rotation, opacity, sizing, and position.
- [Update watermarks]({{< ref "watermark/python-net/developer-guide/basic-usage/update.md" >}}) — find existing watermarks and change them.
- [Clear watermarks]({{< ref "watermark/python-net/developer-guide/basic-usage/clear.md" >}}) — find and remove existing watermarks.
- [Document preview]({{< ref "watermark/python-net/developer-guide/basic-usage/preview.md" >}}) — render document pages as images.
- [Get document info]({{< ref "watermark/python-net/developer-guide/basic-usage/get-document-info.md" >}}) — read file type, page count, and size.
- [Get supported file formats]({{< ref "watermark/python-net/developer-guide/basic-usage/get-supported-file-formats.md" >}}) — list every supported format.

For fine-grained control and format-specific scenarios, continue to the [Advanced usage]({{< ref "watermark/python-net/developer-guide/advanced-usage" >}}) section.
